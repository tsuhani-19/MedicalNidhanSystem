\### Prerequisites

\- Python 3.8+

\- Git



\### Frontend Setup



Clone repository

git clone https://github.com/tsuhani-19/MedicalNidhanSystem.git

cd MedicalNidhanSystem/frontend



Install dependencies

npm install



Configure environment

cp .env.example .env.local

source venv/bin/activate # macOS/Linux

python app.py
